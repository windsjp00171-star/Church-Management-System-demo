# 教學導引 — 聚焦式示範腳本

> 用途：現場 demo / 教學時照著走。**分三種角色**示範，講重點不堆功能。
> 每段格式：**路徑 → 操作 → 講解重點**。教會沒開的模組（見 `HANDOVER.md` 環境變數）直接跳過。
> 建議總時長 30–40 分鐘。

---

## 開場（2 分鐘）

- **路徑**：手機開系統網址 → 點「用 LINE 登入」
- **操作**：用 LINE 登入 → 進入首頁，看功能卡片
- **講解重點**：
  - 「全系統用 **LINE 登入**，會友不用記新帳號密碼。」
  - 「首頁這些卡片就是各功能入口，**哪些要開、名稱叫什麼，後台都能調**。」

---

## 一、會友視角（手機，最重要，先示範這個）

### 1. 報名活動
- **路徑**：首頁 →「活動報名」→ 選一個活動
- **操作**：填表單 → 送出 → 看到「報名成功」
- **重點**：「報名表單欄位是教會自訂的；額滿可候補；也能設付費活動。」

### 2. 活動簽到
- **路徑**：活動當天 → 掃現場 QR Code（或點簽到連結）
- **操作**：一鍵簽到 → 完成
- **重點**：「現場掃碼即可，大螢幕還會跳慶祝動畫炒氣氛。」

### 3. 代禱麥田 *(未開可略)*
- **路徑**：首頁 →「代禱」
- **操作**：發一則代禱 → 對別人的代禱按「🙏」
- **重點**：「弟兄姊妹彼此代禱，可匿名；蒙應允後可標記見證。」

### 4. 天父日記 *(未開可略)*
- **路徑**：首頁 →「天父日記」
- **操作**：選今日讀經進度 → 寫一段 → 看 **AI 反思引導**
- **重點**：「跟著讀經進度寫靈修，AI 會給反思問題幫助默想。」

---

## 二、小組長視角（牧養核心）

### 小組回報
- **路徑**：首頁 →「小組回報」→ 本週回報單
- **操作**：填出席人數 → 逐位組員勾出席狀態 → 填四面向（屬靈/家庭/工作/健康）→ 送出
- **重點**：
  - 「小組長**每週線上回報**，不用再傳紙本或 LINE 訊息。」
  - 「牧者後台能**彙整查閱**所有小組狀況，一目了然。」
  - ⚠️ *這就是這間教會要客製調整的地方，明天依他們實際聚會邏輯改。*

---

## 三、管理員視角（後台，講給窗口/同工聽）

### 1. 建立活動
- **路徑**：`/admin` →「活動管理」→ 新增活動
- **操作**：填基本資料 → 加自訂報名欄位 → 開啟簽到 → 儲存
- **重點**：「自訂欄位、人數上限、候補、白名單、付費都在這設。」

### 2. 即時簽到看板 / 名單匯出
- **路徑**：活動 →「即時簽到」/「匯出名單」
- **重點**：「同工現場即時看到誰到了；事後一鍵匯出 Excel。」

### 3. 會員管理（超級管理員）
- **路徑**：`/admin` →「會員管理」
- **操作**：搜尋會友 → 設服事標籤 / 權限
- **重點**：「會友、訪客分類，權限分層；**最高權限交給教會主責行政**。」

### 4. 資料備份中心（一定要教！）⭐
- **路徑**：`/admin` →「系統備份 / 交接」→ 資料備份中心（`/admin/backup`）
- **操作**：示範下載 **CSV / 全部打包 ZIP / HTML 離線閱讀器**
- **重點**：
  - 「**即使系統哪天停了，這裡能把所有資料拿回來**，CSV 直接 Excel 開。」
  - 「HTML 離線版雙擊就能看，不需網路、不需系統。」
  - 「這是教會**資料自主**的保險，建議定期下載備份。」

---

## 收尾（2 分鐘）

- 「會友資料、系統最高權限**全程在教會手上**；我只負責程式維護。」
- 「想改首頁、開關功能、調名稱，後台都能做；**要改邏輯的部分我來處理**。」
- 「平台都用免費方案起步，日後要更穩再升級即可。」
