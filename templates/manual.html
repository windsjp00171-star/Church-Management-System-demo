22. 權限說明

👤 一般會友篇

1. 如何登入

  1. 開啟系統網址，點選綠色 「LINE 登入」 按鈕
  2. 系統會跳轉到 LINE 授權頁面，點 「同意」
  3. 完成後自動回到首頁，右上角會顯示你的 LINE 大頭貼
💡 第一次登入會要求填寫基本資料（必填），填完才能進入系統

2. 填寫 / 修改個人資料

  1. 第一次登入後，系統會自動跳到個人資料頁（必須填寫，無法跳過）
  2. 先選擇 身分：
    • 🏠 本教會會友：可看到門訓學程、行事曆、代禱牆等完整功能，需填寫所屬小組
    • 🙋 外部人員 / 訪客：僅顯示活動報名、每日經文等基本功能
  3. 填寫 真實姓名（LINE 暱稱不一定是真名）
  4. 會友請選擇 所屬小組（可多選）和 服事團隊（選填）
  5. 點 「完成，開始使用」
💡 之後想修改資料，點首頁右上角名字 → 個人資料

3. 報名活動

會友報名（LINE 登入）

  1. 首頁「現正開放報名」區塊找到活動，點 「立即報名」
  2. 確認活動資訊後，填寫報名表單
  3. 點 「確認報名」 完成，系統會發送站內通知確認

外部朋友報名（無需 LINE）

部分活動開放外部報名，連結分享後無需 LINE 即可報名：

  1. 開啟報名連結，看到 「你的身分是？」 選擇頁
  2. 點 「我是外部朋友」，進入報名表單
  3. 填寫 姓名 及其他活動需要的資料
  4. 點 「確認報名」，報名完成
⚠️ 注意事項：
• 報名截止時間過後無法再報名
• 部分活動為 限定名單，不在名單內看不到報名按鈕
• 部分活動可代替他人報名（同帳號多筆），代報者填寫被代報者姓名即可

4. 取消報名

  1. 點活動詳情頁
  2. 找到已報名的區塊，點 「取消此筆報名」
  3. 確認後即取消（已簽到者無法取消）

5. 門訓學程

  1. 首頁「現正開放報名」區找到門訓課程，點 「立即報名」
  2. 確認前置學程資格後完成報名
  3. 報名後系統會自動發站內通知確認，包含費用資訊
  4. 每堂課結束後由管理員記錄出席
  5. 完課後管理員會在系統標記，可在 「我的記錄」 查看進度
📚 部分學程需先完成前置學程才能報名；
初次報名者教材為必購，已完訓的回鍋學員可自行選擇是否購買

6. 我的記錄

  1. 點上方導覽列的 「我的記錄」
  2. 切換兩個 Tab 查看：
    • 活動報名：所有報名過的活動、簽到狀態、付款狀態
    • 門訓學程：所有學程、出席堂次進度條、完課狀態、費用付款狀態

7. 行事曆 / 個人行程

首頁行事曆整合了教會公告事項與個人私人行程，只有你自己看得到自己的行程。

行事曆顏色說明

⛪ 教會行事 🎉 活動 📖 門訓堂次 🙋 個人行程

新增個人行程

  1. 在首頁行事曆點選 任意一天
  2. 下方出現當天行程詳情
  3. 點橘色虛線 「＋ 新增個人行程」 按鈕
  4. 填寫標題與備註（日期已自動帶入），點 「儲存」
🙋 個人行程只有你自己看得到，其他人無法看到你的個人行程

8. 每週週報

  1. 首頁找到 「本週週報」 區塊
  2. 點 「閱讀完整週報」 查看本週週報全文
  3. 點 「查看所有週報」 可翻閱歷史週報
💡 每週由管理員更新，包含本週聚會資訊、代禱事項、重要公告等

9. 每日經文祝福卡

  1. 首頁功能選單點 「🎴 每日經文」
  2. 系統自動抽出今日祝福卡，顯示經文內容、出處與你的名字
  3. 點 「📲 存到手機」 可下載圖片，分享或存為桌布
🎴 每人每天只能抽一次，隔天零點後自動解鎖

10. 🔔 站內通知

右上角的鈴鐺圖示是系統通知中心，會自動接收以下類型通知：

📋 報名成功：報名活動或課程後自動發送，包含時間、費用等確認資訊
💰 繳費提醒：活動/課程前幾天，若有費用尚未繳清，系統會自動提醒
📢 活動通知：管理員針對已報名者發送的重要通知（如場地異動）
📣 全體公告：超級管理員發送的全體廣播公告
  1. 點右上角 🔔 鈴鐺，展開通知面板
  2. 紅色數字代表未讀通知數量
  3. 點任一通知可跳轉到相關頁面，並標記為已讀
  4. 點 「全部已讀」 一次清除所有未讀

11. 🌾 代禱牆

點導覽列 「🌾 代禱」 進入代禱牆，這裡可以提出代禱需求、為弟兄姊妹代禱回應。
每則代禱就像一粒麥子，隨著大家的回應一起成長。

麥子成長階段

🌰 種子：剛發出，0 回應 🌱 發芽：1–3 人回應 🌿 生長：4–9 人回應 🌾 成熟：10+ 人回應 ✨🌾 蒙應允：禱告蒙神垂聽

發出代禱

  1. 點金色 「🙏 發出代禱」 按鈕
  2. 填寫標題、詳細內容（選填）、選擇類別
  3. 可選擇 具名 或 匿名 發出
  4. 點 「🌾 播下這粒種子」

為他人代禱

  • 點卡片下方的表情符號（🙏 ❤️ 🕯️ ✨ 💪）表達代禱支持
  • 點 「💬 回應」 展開留言區，用文字鼓勵
  • 留言也可選擇匿名

標記蒙應允

  • 當禱告蒙神垂聽，點自己代禱卡片上的 「✨ 蒙應允」
  • 卡片會發出金光，記錄這份恩典 ✨

12. 加到手機主畫面（桌面捷徑）

加完之後像 App 一樣點圖示開啟，不用每次從 LINE 找連結

⚠️ 重要：從 LINE 開啟時，LINE 內建瀏覽器沒有「加到主畫面」功能，
必須先切換到手機的預設瀏覽器（Chrome 或 Safari）才能操作。
Android Chrome 操作步驟
  1. 在 LINE 中開啟系統連結後，點右上角 「⋯」
  2. 點 「用其他應用程式開啟」→ 選 Chrome
  3. Chrome 開啟後，點右上角 「⋮」（三個點）
  4. 點 「新增至主畫面」
  5. 名稱保持「{{ church_short_name }}」，點 「新增」 完成
💡 部分 Android 手機 Chrome 會在網址列下方自動出現「新增到主畫面」橫幅，直接點就好
iPhone Safari 操作步驟
  1. 在 LINE 中開啟系統連結後，點右下角 「⋯」
  2. 點 「在 Safari 中開啟」
  3. Safari 開啟後，點畫面底部中間的 分享按鈕（方框加箭頭 ↑）
  4. 往下滑找到 「加入主畫面」，點一下
  5. 名稱保持「{{ church_short_name }}」，點右上角 「新增」 完成
💡 iPhone 必須用 Safari，Chrome for iOS 目前不支援加到主畫面

13. 建立 / 編輯活動

建立新活動

  1. 後台 → 「活動管理」 → 右上角 「新增活動」
  2. 填寫活動基本資訊（名稱、地點、時間、人數上限、費用）
  3. 設定 ⏰ 繳費提醒天數（預設 3 天前，有費用且未繳才會通知）
  4. 設定 報名開放時間（可設定自動開始/截止）
  5. 上傳 活動海報（選填）
  6. 新增 自訂報名欄位（見下方說明）
  7. 點 「儲存活動」

自訂報名欄位類型

類型 說明
文字單行文字輸入
長文字多行文字輸入（適合備注）
單選多個選項只能選一個，可加「其他」開放填寫
多選可勾選多個選項，可加「其他」開放填寫
下拉選單下拉選擇，可設定跳題條件
數字只能輸入數字
日期日期選擇器
💡 跳題條件：設定「只在特定答案時才顯示此問題」，例如選了「需要接送」才顯示地址欄位

進階設定

  • 同帳號重複報名：允許同一帳號幫多人報名（代報）
  • 開放外部報名：讓沒有 LINE 帳號的朋友也能用姓名報名；報名頁會先詢問身分，選「外部朋友」直接填表，無需登入
  • 限定白名單報名：只有名單內的人才能報名
  • 啟用電子簽到：產生 QR Code 供現場掃描報到
💡 代報小提醒：同帳號幫多人報名時，建議在報名表中加入「姓名」欄位讓報名者填寫被代報者的名字，系統簽到名單會以此欄位顯示，而非帳號持有人的名字

14. 管理報名名單

  1. 後台 → 活動管理 → 點 「名單」
  2. 可看到所有報名者姓名、填寫答案、簽到狀態
  3. 有設定費用的活動可手動切換 付款狀態：
    ⬜ 未付款 → ✅ 已付款 → 🔵 免收費（點按鈕循環切換）
  4. 右上角 「📢 發送通知」：針對已報名者發送活動通知
  5. 右上角 「匯出 Excel」：下載完整名單

✏️ 補填 / 修改報名資料

  1. 每筆報名右側有 ✏️ 按鈕
  2. 點開右側滑入面板，可補填或修改所有自訂欄位答案
  3. 點 「儲存」 完成
💡 若活動啟用白名單，名單頁下方可新增/移除報名資格名單

15. 電子簽到

設定簽到

  • 建立活動時勾選 「啟用電子簽到」
  • 選擇模式：
僅限已報名者：只有事先報名的會友能掃碼簽到
全開放：任何人掃碼都能簽到，系統自動建立到場紀錄
⚡ 掃碼即自動簽到：會友掃碼後不需點確認，直接完成簽到。
適合核心同工會等場合，一掃即到。
🎉 派對動畫（選填）：勾選後，會友簽到成功時播放彩帶爆炸 + 氣球浮升動畫。
適合週年慶、聖誕、聚會等特別場合。一般聚會可不勾選。

活動當天流程（會友）

  1. 後台活動管理 → 點 「QR」 顯示簽到 QR Code
  2. 投影或列印，讓會友自行掃描
  3. 會友掃碼用 LINE 登入即完成簽到
  4. 後台點 「即時」 查看即時簽到狀況（3 秒自動更新）

外部朋友簽到（開放外部報名的活動）

  1. 外部朋友掃碼後，看到 「你的身分是？」 選擇頁
  2. 點 「我是外部朋友」，進入打碼名單頁
  3. 名單上顯示已報名的外部朋友名字（中間文字打碼，如「王●明」）
  4. 找到自己的名字，點選即完成簽到
💡 候補轉正式報名的外部朋友，同樣會出現在打碼名單，可自行簽到；仍在候補的朋友不在名單中，由同工協助代簽

🎉 現場展示頁（大螢幕 / 平板用）

開啟派對動畫後，「即時簽到狀況」頁右上角會出現紫色 「🎉 現場展示」 按鈕。

  • 點擊後在新分頁開啟，拉到現場電視或平板全螢幕
  • 每 3 秒自動偵測新簽到，有人到場即自動觸發動畫
  • 顯示：歡迎 + 到場者名字（大字） + 彩帶 + 氣球
  • 6 秒後自動恢復待機畫面，顯示目前到場人數
  • 同工代簽時，氣球顯示被簽到者的名字，不是代簽的管理員

16. 教會行事管理

管理員可在行事曆新增教會公告行事，所有會友都看得到。

  1. 首頁行事曆點選目標日期
  2. 點出現的紫色虛線 「＋ 教會行事」 按鈕
  3. 填寫標題、說明、選擇顏色，點 「儲存」
  4. 若需修改：點當天行事旁的 「✏️ 編輯」 或 「🗑 刪除」
💡 教會行事、系統活動、門訓堂次都會顯示在行事曆上

17. 門訓課程管理

建立課程

  1. 後台 → 「門訓學程」 → 「新增學程」
  2. 填寫課程名稱、期數、總堂次、完課門檻
  3. 可設定 教材費 與 ⏰ 繳費提醒天數
  4. 若有前置學程，選擇必須先完成的課程
  5. 建立後進入課程詳情，新增各堂次（日期、標題）

出席名單管理

  • 名單頁「本期出席」Tab：點每堂格子手動標記出席
  • 教材欄位：點擊 ✓/— 切換該學員是否需要購買教材
  • 💰 付款欄：點按鈕循環切換 ⬜ 未付款 → ✅ 已付款 → 🔵 免收費
  • 達標者可 「✅ 一鍵完訓達標者」 批次處理
  • 「新增學員」Tab：搜尋後點 「加入」，可勾選直接標記完訓

名單顯示格式

  • 顯示格式為：真實姓名（LINE 暱稱）
  • 🔄 回鍋 標籤：曾完訓後再次報名的學員

小組完課總覽

  • 後台 → 「門訓總覽」：依小組分區顯示每人各課程狀態
  • 支援列印，方便與牧者/小組長確認

18. 發送通知 / 廣播公告

針對已報名者發送通知

  1. 活動或課程的名單頁 → 點右上角 「📢 發送通知」
  2. 填寫通知內容（如場地異動、注意事項）
  3. 點 「📢 發送」，所有已報名者的鈴鐺會收到通知

繳費提醒（自動）

⏰ 系統會在活動/課程開始前 N 天（你在表單設定的天數，預設 3 天），
自動提醒有費用尚未繳清的學員，已付款者不會收到提醒

📣 全體廣播公告（超管限定）

  1. 後台 → 「📢 公告廣播」（需超級管理員）
  2. 填寫公告標題、內容
  3. 點 「發送全體公告」，所有使用者都會收到站內通知

19. 小組標籤管理

小組標籤分兩種，影響會友資料填寫和門訓總覽分組：

類型 說明 範例
主標籤 會友所屬的週間小組，門訓總覽依此分組 恩典組、喜樂組
副標籤 服事團隊（選填），不影響門訓分組 敬拜團、招待
💡 後台 → 「小組管理」可新增、修改、調整排序、切換主/副標籤（需超級管理員）

20. 用戶管理 / 訪客升級為會友

當外部人員成為會友，管理員可手動升級身分：

  1. 後台 → 「用戶管理」（需超級管理員）
  2. 上方 Tab 可篩選：全部 / 🏠 會友 / 🙋 訪客
  3. 搜尋姓名 → 點 「編輯資料」
  4. 身分從「訪客 / 外部人員」改為 「本教會會友」
  5. 選擇所屬小組後儲存，對方下次登入即自動看到完整功能

快捷連結的訪客顯示控制

  • 後台 → 「快捷連結管理」 → 編輯連結 → 勾選 「僅限會友」
  • 勾選後訪客看不到，會友才看得到

21. 小組討論管理

設定每週在首頁顯示的影片連結與討論問題，供小組聚會使用。

  1. 後台 → 「小組討論」
  2. 點 「＋ 新增」，填寫標題、YouTube 連結、顯示日期區間
  3. 依序輸入討論問題，按 Enter 或「新增問題」加入
  4. 勾選「啟用」後，在指定日期範圍內會自動顯示在首頁
💡 日期範圍到期後會自動停止顯示，不需手動關閉。可同時建立多份排程，系統依日期自動切換。
