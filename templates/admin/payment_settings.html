❓ 怎麼申請金流帳號？點此展開教學
🟢 綠界科技 ECPay（推薦台灣教會）
  1. 前往 ecpay.com.tw → 右上角「立即申請」→ 選「一般特店」
  2. 填寫教會基本資料（統編 / 負責人）→ 等審核（約 1–3 個工作天）
  3. 審核通過後，登入綠界後台 → 系統開發管理 → 介接參數設定
  4. 複製「特店編號 (MerchantID)」、「HashKey」、「HashIV」填入下方
  5. 先選「測試環境」試跑，確認付款流程無誤後再切換「正式環境」
💡 測試用特店編號：2000132，HashKey：5294y06JbISpM5x9，HashIV：v77hoKGq4kWxNNIS（不會真實扣款）
💚 LINE Pay
  1. 前往 pay.line.me/tw → 商家申請 → 填寫商業登記資料
  2. 審核通過後進入 LINE Pay 商家後台
  3. 左側選單 → 管理付款連結 → 連結資訊
  4. 複製「Channel ID」與「Channel Secret Key」填入下方
  5. 先選「Sandbox 沙盒模式」測試，確認後切換正式環境
💡 申請 LINE Pay 需要公司統編或商業登記，個人教會可考慮優先使用綠界。
📝 手動繳費說明（最簡單）

不需申請任何帳號。選「手動繳費說明」後，在文字框填入教會的匯款帳號或其他繳費方式， 會友點「前往繳費」時只會看到這段說明文字。
適合暫時沒有金流帳號、或只想先提供匯款資訊的情況。

📝 繳費說明內容

這段文字會顯示在報名者的「去繳費」頁面，例如銀行帳號、匯款說明、繳費截止等。

🟢 綠界 ECPay 設定

💡 到 綠界後台 → 系統開發管理 → 介接參數設定 取得以下資料。 測試模式使用綠界的「特店測試帳號」，不會有真實扣款。

💚 LINE Pay 設定

💡 到 LINE Pay 商家後台 → 管理付款連結 → 連結資訊 取得 Channel ID 和 Channel Secret Key。 Sandbox 模式不會有真實扣款。

🧾 手續費由誰承擔？

綠界 / LINE Pay 每筆交易會收取平台手續費（信用卡約 2.75%，ATM NT$15，超商 NT$25–30）。
選擇由教會吸收或讓繳費者多付。
