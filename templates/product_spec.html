1. 系統概覽

🎯

定位與目標

  • 整合活動報名、電子簽到、小組回報、檔案分享、靈修日記、門訓課程等教會日常行政需求
  • 以 LINE 帳號為唯一登入入口，降低中高齡使用者的操作門檻
  • 支援多身分（會友 / 小組長 / 同工 / 牧者 / 管理員）動態顯示對應功能
  • 全端 Mobile-first 設計，主要使用場景為手機瀏覽器及 LINE 內建瀏覽器
📦

3. 功能模組說明

🔑

LINE 登入

/auth/login
  • LINE Login OAuth2 流程，不儲存密碼
  • 在 LINE 內建瀏覽器自動呼叫 LIFF SDK 靜默登入；外部瀏覽器直接顯示登入按鈕，無等待時間
  • 管理員 LINE User ID 透過環境變數 ADMIN_LINE_USER_IDS 設定，登入時自動賦予管理員權限
  • 訪客首次登入後引導填寫真實姓名，填完升格為會友
🎉

活動報名與電子簽到

/events  /checkin
  • 管理員可建立活動並設計自訂報名表（文字、單選、複選、條件顯示欄位）
  • 支援付費活動、候補名單、開放式報名（不限報名者）、白名單模式
  • 外部朋友（非 LINE 登入）可透過獨立報名頁參加
  • 電子簽到提供 QR Code 掃描、手動搜尋補簽、Kiosk 現場自助簽到三種模式
  • 即時簽到監控介面（3 秒自動更新），可匯出出席名單
📋

小組回報與聚會統計

/cell-report
  • 小組長每週填寫：出席人數、信仰進度（靈修/讀經/禱告）、聚會主題、代禱事項
  • 支援標記「本週無聚會」並填寫原因
  • 聚會人數回報：成人主日（多堂）、兒童主日、禱告會、晨禱
  • 自訂聚會類型 NEW：管理員可在「聚會設定」新增任意聚會（如青年崇拜），自動出現在 Dashboard 統計卡片與填報連結
  • Dashboard（同工 / 牧者版）依週次顯示所有小組回報狀況與聚會人數總覽，可切換歷史週次
📔

天父日記（靈修日記）

/diary
  • 依讀經計畫自動導入當日經文，會友記錄靈修心得
  • AI 反思引導（Groq 主、Gemini 備援），逐字串流回應，10 秒超時自動重試
  • 日記可設定加密鎖定；授權牧者後可供牧養關顧查閱
  • 牧者查閱介面可篩選日期、搜尋關鍵字
📁

檔案分享

/files  /folders
  • 上傳至 Cloudflare R2，支援批次上傳、拖拉上傳、拍照上傳
  • 權限控制：公開 / 私人 / 指定會員 / 指定小組 / 指定角色
  • 支援密碼保護、改名、資料夾管理
  • 單檔上傳上限 500 MB，大型檔案顯示進度條
📚

門訓課程

/courses
  • 課程建立：設定先修條件、報名期限、容量、講義上傳
  • 出席追蹤：每堂課掃 QR Code 點名，自動計算出席率
  • 完訓認證：達標後核發完訓記錄，可批次發放
  • 後台管理名單、手動加退選
✨

其他功能

  • 代禱麥田：會友發布代禱事項，他人可回覆並標記蒙應允
  • 行事曆：個人報名活動 + 教會行事一覽
  • 每週週報：管理員上傳 PDF，首頁資訊列顯示
  • 福音探索：外部朋友留詢問，同工跟進管理
  • 每日經文卡：生成精美圖卡可下載/分享
  • 站內通知：活動開始提醒、候補錄取通知等
  • 設定精靈：/setup-wizard 引導首次部署的環境變數與資料表建立確認

5. 部署與環境變數

🚀

快速部署流程

  • Fork 此 repository → Render 選「New Web Service」連結 repo
  • 設定以下環境變數後，首次啟動系統會顯示 /setup-wizard 設定精靈
  • 在 Supabase SQL Editor 執行 /setup-wizard 頁面提供的建表 SQL
  • 確認所有必填環境變數與資料表均為綠色 ✓ 後即可上線
⚙️

環境變數一覽

變數名稱必填說明
FLASK_SECRET_KEY✓Session 加密金鑰（用 secrets.token_hex(32) 生成）
LINE_CHANNEL_ID✓LINE Login Channel ID
LINE_CHANNEL_SECRET✓LINE Login Channel Secret
LINE_REDIRECT_URI✓OAuth 回呼網址（https://你的網域/auth/callback）
SUPABASE_URL✓Supabase 專案 URL
SUPABASE_KEY✓Supabase anon key
CHURCH_NAME選填教會全名（顯示於標題列）
ADMIN_LINE_USER_IDS選填管理員 LINE User ID，逗號分隔
LINE_LIFF_ID選填LIFF App ID（LINE 內自動登入用）
R2_ENDPOINT / R2_ACCESS_KEY_ID / R2_SECRET_ACCESS_KEY / R2_BUCKET_NAME選填Cloudflare R2 檔案分享模組
GROQ_API_KEY / GEMINI_API_KEY選填AI 反思引導（天父日記）
